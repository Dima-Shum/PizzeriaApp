# 🍕 Пиццерия - Мобильное приложение на .NET MAUI

Приложение для заказа пиццы с конструктором, корзиной и оформлением заказа.

## 📱 Функциональность

- **Меню пицц** — просмотр списка пицц с описанием и ценами
<img width="957" height="691" alt="image" src="https://github.com/user-attachments/assets/56577449-11a8-4762-bee4-42255b35c49b" />

- **Конструктор** — выбор размера, добавление ингредиентов, расчёт стоимости
<img width="956" height="929" alt="image" src="https://github.com/user-attachments/assets/99561f08-8ab6-4d23-9151-5e28269f44d7" />

- **Корзина** — управление заказом, изменение количества, удаление позиций
<img width="955" height="1028" alt="image" src="https://github.com/user-attachments/assets/7c235b1c-b8ba-47d7-9519-e9a06cd542ae" />

- **Оформление** — ввод данных клиента, валидация, имитация отправки заказа
<img width="955" height="979" alt="image" src="https://github.com/user-attachments/assets/41cf4e44-822d-45e3-b5b9-256699191e61" />

## 🛠 Технологии

- **.NET MAUI** — кроссплатформенная разработка
- **MVVM** — архитектурный паттерн
- **JSON** — хранение данных меню
